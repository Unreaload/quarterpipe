TeamUp Cheat Sheet quarterpipe.hamburg — Anleitung fuer den Veranstaltungskalender

Wie die Website Events anzeigt

  • Zeitraum: 7 Tage zurueck bis 30 Tage voraus
  • Aktualisierung: stuendlich (Cache von 1 Stunde)
  • Vergangene Events werden ausgegraut dargestellt
  • Events mit Startzeit zeigen Datum + Uhrzeit
  • Ganztagesevents zeigen nur das Datum (keine Uhrzeit)

Bild hinzufuegen

  1. Event oeffnen → Anhang hinzufuegen (Bueroklammer-Icon)
  2. Bilddatei hochladen (jpg, png, webp, gif, avif oder svg)
  3. Das erste Bild wird auf der Website angezeigt
  4. Weitere Anhaenge (PDFs etc.) werden ignoriert
Wichtig: Bilder als Anhang hochladen, nicht in die Kommentare posten!

Beispiel: vollstaendiges Event

Titel: Konzert: Speedway Start: 28.05.2026, 20:00 Ende: 28.05.2026, 23:00 Location: Hardcore Punk aus Hamburg Notes: Support: TBA. Einlass ab 19:30, Beginn 20:00. VVK 12 EUR / AK 15 EUR Veranst.: extern Anhang: speedway-flyer.jpg

Event soll NICHT auf der Website erscheinen

Weg 1: Veranstalter*in auf „geschlossen“ setzen

Events werden automatisch ausgeblendet, wenn das Custom-Feld Veranstalter*in einen Wert mit geschlossen enthaelt:

  • extern_geschlossen — externe geschlossene Veranstaltung (z.B. Geburtstage)
  • amigo_geschlossen — interne geschlossene Veranstaltung

Weg 2: Filterwoerter im Titel

Events werden zusaetzlich ausgeblendet, wenn Titel oder Location eines dieser Woerter enthalten (Gross-/Kleinschreibung egal):

  • Vorlage Veranstaltung
  • Anfrage

Ganztagesevents — Achtung

  • Ganztagesevents funktionieren, zeigen aber keine Uhrzeit an
  • Fuer Events mit Einlasszeit: kein Ganztagesevent verwenden, sondern konkrete Start-/Endzeit setzen
  • Ganztagesevents eignen sich fuer mehrtaegige Festivals, Aktionstage etc.

Schnellreferenz

sichtbar auf Website

  • Titel ohne Filterwoerter
  • Veranstalter*in: amigo_ oder extern
  • Startzeit setzen (kein Ganztagesevent)
  • Location = Untertitel
  • Notes = Beschreibung (aufklappbar)
  • Anhang = Eventbild (erstes Bild zaehlt)

versteckt auf Website

  • Veranstalter*in enthaelt „geschlossen“
    (extern_geschlossen / amigo_geschlossen)
  • „Anfrage“ im Titel oder Location
  • „Vorlage Veranstaltung“ im Titel oder Location
